## Assignment Description

This assignment implements a C++ program that accepts a list of numbers
from the user and sorts them in **ascending order** using the **Bubble
Sort algorithm**.

The program prompts the user to enter the number of elements and then
input the elements. After sorting, the program displays the numbers
arranged from the smallest to the largest.

------------------------------------------------------------------------

## Algorithm Used

The program uses the **Bubble Sort algorithm**.

### Bubble Sort Overview

Bubble Sort repeatedly compares adjacent elements in an array and swaps
them if they are in the wrong order. This process continues until the
list is fully sorted.

### Steps

1.  Input the number of elements.
2.  Store the elements in an array.
3.  Compare adjacent elements.
4.  Swap them if the first element is greater than the second.
5.  Repeat the process until the array is sorted.
6.  Display the sorted numbers.
