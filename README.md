## Project Description
This assignment implements a C++ program that sorts numbers in **ascending order** using a function called **PROXMAN**.

The **PROXMAN function** is responsible for sorting an array of numbers from the smallest to the largest value. The algorithm works by repeatedly comparing adjacent elements and swapping them if they are in the wrong order until the array becomes fully sorted.

The program asks the user to enter the number of elements, then input the numbers, after which the program sorts them and displays the results in ascending order.

---

## Algorithm Used: PROXMAN

The **PROXMAN algorithm** is implemented as a sorting function in the program. It is based on the **Bubble Sort technique**, where elements are repeatedly compared and swapped.

### Steps of the Algorithm
1. Input the number of elements.
2. Store the elements in an array.
3. Compare adjacent elements in the array.
4. Swap the elements if the first is greater than the second.
5. Repeat the process until no more swaps are needed.
6. Display the sorted array.
